# NeuroEstate AI

NeuroEstate AI is a machine learning web application that predicts house prices based on property features such as square feet, bedrooms, bathrooms, neighbourhood, and house type.

The project combines Machine Learning with a modern web interface to deliver fast and accurate property price predictions.

---

## Features

- AI-based house price prediction
- Modern responsive UI
- Flask backend integration
- Machine Learning model using Scikit-learn
- Real-time prediction system
- End-to-end ML project deployment

---

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- Flask

### Machine Learning
- Pandas
- NumPy
- Scikit-learn
- Random Forest Regressor
